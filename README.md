# ТБС

Многостраничный сайт-портал Тульской Библиотечной Системы

## Функционал

- Адаптивная, валидная и кросбраузерная верстка
- Слайдеры
- Модальное окно
- Мобильное меню
- Yandex Map

## Стек технологий

- HTML
- HTML шаблонизатор 
- CSS
- SASS
- JS
- Сборщик Webpack + Gulp
- Swiper Slider
- Yandex Map Api
